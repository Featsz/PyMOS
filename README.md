# PyMOS — Python Mini-OS

Hey! I built this terminal emulator from scratch on my phone.  
It's not a real OS, but it looks and feels like one.

## What can it do?
- Navigate folders (`cd`, `ls`, `pwd`)
- Create and delete files (`fm`, `mkdir`, `rmdir`, `rm`)
- Read file content (`cat`)
- Show system info (`sys`, `sysfetch`)
- Custom prompt (change it in `config.json`)

## Install
pip install colorama requests whatismyip
python pymos.py
